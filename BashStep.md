-   

    All Implemented Interfaces:

    ------------------------------------------------------------------------

        public class BashStep
        extends ShellStep

    ::: block
    Command that makes it possible to run an arbitrary command in Bash.
    Whenever possible, a more specific subclass of
    [`ShellStep`](ShellStep.html "class in com.facebook.buck.shell")
    should be preferred. BashCommand should be reserved for cases where
    the expressiveness of Bash (often in the form of \*-shell-expansion)
    makes the command considerably easier to implement.
    :::
