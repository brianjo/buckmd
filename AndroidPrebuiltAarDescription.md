    ::: block
    Description for a
    [`BuildRule`](../core/rules/BuildRule.html "interface in com.facebook.buck.core.rules")
    that wraps an `.aar` file as an Android dependency.
    This represents an Android Library Project packaged as an `.aar`
    bundle as specified by:
    https://developer.android.com/studio/projects/android-library#aar-contents.
    When it is in the packageable deps of an
    [`AndroidBinary`](AndroidBinary.html "class in com.facebook.buck.android"),
    its contents will be included in the generated APK.

    Note that the `aar` may be specified as a
    [`SourcePath`](../core/sourcepath/SourcePath.html "interface in com.facebook.buck.core.sourcepath"),
    so it could be either a binary `.aar` file checked into version
    control, or a zip file that conforms to the `  .aar` specification
    that is generated by another build rule.
