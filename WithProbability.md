## Interface WithProbability {#interface-withprobability .title title="Interface WithProbability"}
:::

::: contentContainer
::: description
-   

    All Known Implementing Classes:
    :   `AndroidInstallConfig.ConcurrentInstall`, `CommonGroups`,
        `DepsAwareExecutorType`

    ------------------------------------------------------------------------

        public interface WithProbability

    ::: block
    Marks an enum for toggling buck experiments as having probability.
    This means that each itemization of the enum can be attached with a
    specific probability of it being selected by a random trial.
    :::
