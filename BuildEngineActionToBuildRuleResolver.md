        public class BuildEngineActionToBuildRuleResolver
        extends Object

    ::: block
    Resolves a
    [`BuildEngineAction`](../BuildEngineAction.html "interface in com.facebook.buck.core.build.action")
    into the corresponding
    [`BuildRule`](../../../rules/BuildRule.html "interface in com.facebook.buck.core.rules").
    This only exists due to legacy compatibility reasons so that we can
    incrementally migrate the build engine to use
    [`BuildEngineAction`](../BuildEngineAction.html "interface in com.facebook.buck.core.build.action")
    :::
