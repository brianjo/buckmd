## Interface CompilationUnitTreeProxy {#interface-compilationunittreeproxy .title title="Interface CompilationUnitTreeProxy"}
:::

::: contentContainer
::: description
-   

    All Known Implementing Classes:
    :   `CompilationUnitTreeProxyImpl`

    ------------------------------------------------------------------------

        public interface CompilationUnitTreeProxy

    ::: block
    `CompilationUnitTree` is included with the compiler and is thus not
    directly accessible from within Buck\'s class loader. This interface
    is used to identify objects that serve as proxies for
    `CompilationUnitTree`\'s within Buck\'s class loader. These are
    translated back into the underlying `CompilationUnitTree` when
    passed back into the compiler.
    :::
