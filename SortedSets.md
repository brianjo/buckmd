            A view merging the two underlying sorted sets.
            This View performs all operations lazily, and sacrifices CPU
            to reduce memory overhead. If operations are being
            repeatedly performed, it may be better to perform eager
            copies using something like
            `ImmutableSortedSet.copyOf(E[])`.

            The behavior of this view is unspecified if the underlying
            SortedSets are modified after construction.
