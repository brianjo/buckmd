## Interface HasBrokenInputBasedRuleKey {#interface-hasbrokeninputbasedrulekey .title title="Interface HasBrokenInputBasedRuleKey"}
:::

::: contentContainer
::: description
-   

    ------------------------------------------------------------------------

        public interface HasBrokenInputBasedRuleKey

    ::: block
    Used to indicate that this Buildable\'s input-based key doesn\'t
    properly reflect all the inputs to this rule. This shouldn\'t be
    necessary and if used is likely an indication of a bug in either the
    ModernBuildRule framework or this Buildable\'s implementation.
