        -   #### DIRECT_TO_JAR

                public static final JavacOptions.SpoolMode DIRECT_TO_JAR

            ::: block
            Writes the compiler output directly to a .jar file while
            retaining the intermediate .class files in memory. If
            `JavaLibraryDescription.AbstractJavaLibraryDescriptionArg`
            postprocessClassesCommands are present, the builder will
            resort to writing .class files to disk by necessity.
            :::

        []{#INTERMEDIATE_TO_DISK}

                public static final JavacOptions.SpoolMode INTERMEDIATE_TO_DISK

            ::: block
            Writes the intermediate .class files from the compiler
            output to disk which is later packed up into a .jar file.
            :::
