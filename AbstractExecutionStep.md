        public abstract class AbstractExecutionStep

    ::: block
    Abstract implementation of
    [`Step`](Step.html "interface in com.facebook.buck.step") that takes
    the description as a constructor parameter and requires only the
    implementation of
    [`execute(ExecutionContext)`](#execute(com.facebook.buck.core.build.execution.context.ExecutionContext)).
    This facilitates the creation of an anonymous implementation of
    [`Step`](Step.html "interface in com.facebook.buck.step").
